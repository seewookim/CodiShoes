# 👟 CodiShoes

### AI 기반 개인 맞춤형 신발 추천 & 코디 관리 Android 애플리케이션

> “오늘 코디에 가장 잘 어울리는 신발은 무엇일까?”  
CodiShoes는 사용자의 스타일, 상황, 선호도를 분석하여  
**AI 기반으로 신발을 추천하고 코디를 관리할 수 있는 Android 애플리케이션**입니다.

---

## 📌 프로젝트 개요

- **프로젝트명**: CodiShoes  
- **플랫폼**: Android  
- **개발 목적**:  
  딥러닝 및 AI 기반 추천 로직을 실제 모바일 애플리케이션에 적용하여  
  사용자 맞춤형 코디 추천 서비스를 구현

- **핵심 키워드**  
  `AI 추천 시스템` · `딥러닝 기반 의사결정` · `프롬프트 설계` · `Android App`

---

## ✨ 주요 기능

### 1️⃣ AI 기반 신발 추천
- 사용자의 선택 정보(스타일, 색상, 가격대, 상황 등)를 입력값으로 활용
- AI 추천 엔진이 조건을 종합 분석하여 **최적의 신발 목록 추천**

### 2️⃣ 코디 저장 및 관리
- 추천 결과를 **코디로 저장**
- 저장된 코디 목록 확인
- 즐겨찾기(찜) 기능 제공

### 3️⃣ 찜한 신발 관리
- 관심 있는 신발을 찜 목록으로 저장
- 별도의 찜 목록 화면 제공

### 4️⃣ 쇼핑몰 이동 기능
- 추천된 신발 클릭 시 외부 쇼핑몰로 이동 가능

---

## 🧠 AI / 딥러닝 학습 및 적용 내용

### 🔹 AI 학습 목적
- 단순 규칙 기반 추천이 아닌  
  **사용자 선택 패턴을 학습한 추천 로직 설계**
- 딥러닝 개념을 실제 서비스 로직에 적용하는 경험

### 🔹 학습 및 설계 방식
- 사용자의 입력 조건을 **특징 벡터(feature)** 로 정의
  - 스타일 (캐주얼 / 스트릿 / 포멀 등)
  - 색상 선호
  - 가격대
  - 착용 상황 (일상 / 운동 / 모임 등)

- 조건 조합에 따른 추천 결과를 학습 데이터 형태로 설계
- AI 추천 엔진(`RecommendationEngine`)에서  
  가중치 기반 판단 로직을 통해 **가장 적합한 신발을 도출**

> 본 프로젝트에서는  
> **딥러닝 모델 구조 설계 개념 + AI 의사결정 흐름**을  
> Android 환경에 맞게 단순화하여 구현

---

## 🧩 사용한 프롬프트 설계 내용

### 🔹 프롬프트 설계 목적
- AI가 **사용자의 상황과 의도를 정확히 이해**하도록 유도
- 일관된 추천 결과를 얻기 위한 구조화된 입력 설계

---

### 🔹 프롬프트 구조 예시

- text

- '사용자의 스타일은 캐주얼이며,
- 주요 색상은 흰색과 검정색,
- 가격대는 중저가,
- 착용 목적은 일상 활동이다.

- 위 조건에 가장 적합한 신발 스타일을 추천하라.'

### 🔹 프롬프트 설계 전략

- 1.조건 명시형 구조

- 스타일 / 색상 / 가격 / 상황을 명확히 구분

- 2.불필요한 자연어 최소화

- AI 판단에 필요한 핵심 정보만 제공

- 3.일관된 입력 포맷 유지

- 추천 결과의 안정성과 재현성 확보

### 🔹 프롬프트 적용 영역

- 신발 추천 로직 설계

- 코디 요약 문구 생성

- 추천 결과 설명 문장 구성
