# 👟 CodiShoes  
### AI 기반 개인 맞춤형 신발 추천 & 코디 관리 Android 애플리케이션

> *“오늘 코디에 가장 잘 어울리는 신발은 무엇일까?”*  
> CodiShoes는 사용자의 스타일과 조건을 분석해  
> **가장 합리적인 신발 선택을 돕는 AI 코디 어시스턴트 앱**입니다.

---

## 📌 프로젝트 개요

**CodiShoes**는 사용자가 선택한 스타일과 신발 조건을 바탕으로  
AI 추천 로직을 통해 어울리는 신발을 제안하고,  
추천 결과를 **저장·관리·즐겨찾기**까지 할 수 있는  
**종합 신발 추천 Android 애플리케이션**입니다.

단순한 리스트 추천이 아닌,  
> “왜 이 신발이 어울리는지”  

를 **자연스러운 AI 설명 문장**으로 제공하여  
사용자의 선택 부담을 줄이는 것을 목표로 합니다.

---

## 🎯 개발 배경 및 목적

### 🔍 문제 인식
- 신발 선택 시 스타일·가격·용도 고려가 복잡함
- 추천 앱 대부분이 단순 필터 기반
- 추천 이유가 없어 사용자가 신뢰하기 어려움

### 🎯 해결 목표
- 스타일 기반 신발 추천 자동화
- AI 설명을 통한 추천 신뢰도 향상
- 개인화된 코디 저장 및 관리 기능 제공
- 실제 쇼핑 흐름까지 연결되는 UX 설계

---

## ✨ 핵심 기능 요약

| 구분 | 기능 설명 |
|---|---|
| 👟 신발 추천 | 스타일 기반 상위 3개 신발 자동 추천 |
| 🤖 AI 설명 | 추천 이유를 자연어 문장으로 제공 |
| ⭐ 찜 기능 | 마음에 드는 신발 즐겨찾기 |
| 💾 코디 저장 | 추천 결과를 코디로 저장 |
| 🗂 코디 관리 | 저장한 코디 목록 확인 |
| 🛍 쇼핑 연동 | 네이버 쇼핑 검색 결과 바로 이동 |

---

## 🧠 AI 추천 시스템 설계

### 🔹 추천 로직 구조

1. **입력값 분석**
   - 스타일 (캐주얼 / 스트릿 / 미니멀 / 포멀)
   - 신발 타입 (선택 사항)
   - 가격대 (선택 사항)

2. **조건 필터링**
   - 타입 및 가격대 일치 여부
   - 사용자가 선택한 옵션 우선 반영

3. **점수 기반 추천**
   - 스타일 일치 여부에 가장 높은 가중치 부여
   - 가격대에 따른 보정 점수 적용

4. **상위 결과 추출**
   - 점수 기준 상위 3개 신발 추천

---

### 🔹 AI 추천 이유 생성

추천 결과는 단순 점수가 아닌  
**프롬프트 설계 기반 자연어 문장**으로 출력됩니다.

예시:
> “선택한 스트릿 스타일과 잘 어울리는 디자인이며,  
> 가격대와 활용도 면에서 균형 잡힌 선택입니다.”

- 즐겨찾기한 신발만 별도 관리
